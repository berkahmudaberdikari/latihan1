# Ojek Online Landing Page

## Project Description
This project is a landing page for Ojek Online, a motorcycle taxi service widely used in Indonesia. It serves as an introductory webpage to highlight the services, benefits, and features offered by the Ojek Online service.

## Features
- Responsive design for mobile and desktop users
- User-friendly interface
- Detailed service descriptions
- Call to action buttons for easy access to services
- Integration with Google Maps for service areas
- Contact form for inquiries

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/berkahmudaberdikari/latihan1.git
   ```
2. Navigate into the project directory:
   ```bash
   cd latihan1
   ```
3. Open the `index.html` file in your web browser to view the landing page.

## File Structure

## Usage Guide
- Open the `index.html` file in a web browser to view the landing page. 
- The landing page includes various sections, each detailing different aspects of the Ojek Online service. 
- Use the navigation menu to jump to different sections of the landing page.
- Submit the contact form for any inquiries or feedback.
